# Note

### Note is partially based on [Quasars Notify](https://quasar.dev/quasar-plugins/notify)

Examples:

```js
import { note } from '@waelio/utils/dist/note';

note.success('Hi');
note.info('Hi');
note.warning('Hi');
note.error('Hi');

// Log
note.log('Hi from console');
note.log('Hi from console');
```
